
###### **Affiliation** [affiliation]
Hier wird im Volltext der Name der Institution oder Organisation, der ein:e Akteur:in angehört, erfasst. Dies kann beispielsweise der Name einer Hochschule, ein Unternehmen, eine Partei, ein Verband usw. sein. Die Affiliation wird nur codiert, wenn sie im Beitrag explizit erwähnt wird, aus(Amts-)Bezeichnung des:der Akteur:in hervorgeht oder allgemein (d. h. ohne weitere Recherche) bekannt ist. 
Wird beispielsweise vom deutschen Bundeskanzler gesprochen, kann bei der Affiliation "Bundeskanzleramt" notiert werden, ebenso wenn in einem Artikel von 2026 Friedrich Merz zu Wort kommt oder in einem Artikel von 2016 Angela Merkel. Wird die Forschungsministerin erwähnt, ist entsprechend das "Bundesministerium für Forschung, Technologie und Raumfahrt" einzutragen. Die Affiliation ist stets so genau wie möglich (idealerweise auch mit Ortsangabe) festzuhalten. 
Wenn die Affiliation nicht erwähnt wird oder anderweitig erkennbar ist, wird in das Feld "NA" eingetragen, das kann z. B. der Fall sein, wenn ein:e Akteur:in nur als "Künslter:in" oder "Forscher:in" vorgestellt wird.

###### **Geschlecht** [gender]
In dieser Variable wird das Geschlecht der Akteur:in erfasst. Dabei gelten der Name und/oder die Ansprache (z. B. Herr/Frau; er/sie/xier) des:der Akteur:in im Artikel als ausschlaggebend. Ist die Zuordnung zu einem Geschlecht auf dieser Grundlage nicht eindeutig möglich oder weicht sie von der heteronormativen Binarität ab, wird "99 = anderes / nicht eindeutig bestimmbar" codiert.
