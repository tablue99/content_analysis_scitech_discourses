
###### **Bezeichnung** [intro]

In dieser Variable wird erfasst, wie ein:e Akteur:in in einem Artikel bei erstmaliger Nennung vorgestellt wird, d. h. welche Beschreibung und/oder Bezeichnung bei seinem:ihrem erstmaligen Auftreten für ihn:sie neben der namentlichen Nennung gewählt wird. Da oftmals mehrere Beschreibungen auf einmal zur erstmaligen Vorstellungen eines:einer Akteur:in in einem Text gewählt werden (z. B. **Prof.** Christian Drosten ist **Mediziner am Berliner Charité-Klinikum**; Ilya Sutskever, **Experte** für die **Weiterentwicklung von Künstlicher Intelligenz** bei **OpenAI**), ist eine Mehrfachauswahl in dieser Variablen möglich. Werden außer des Namens keine weiteren Spezifizierungen vorgenommen, wird "0 = nur Name" codiert. In diesem Fall ist keine Mehrfachauswahl sinnvoll und daher auch nicht erlaubt.\
**WICHTIG**: Es werden nur die Bezeichnungen und Beschreibungen berücksichtigt, die bei der allerersten Nennung des:der Akteur:in im Text vorkommen.

<p><div style="margin-left:20px;"><b>1</b> Zuordnung zu einer Institution / Affiliation </br>
Die Institution, an der ein:e Akteur:in tätig ist, wird genannt.</br>
Wird für individuelle Akteur:innen keine Institution, aber ein Wohnort genannt, werden diese auch mit diesem Code versehen.</br>
Beispiel: Christian Drosten von der Berliner Charité</div></p>
<p><div style="margin-left:20px;"><b>2</b> Berufsbezeichnung </br>
Der Beruf, den ein:e Akteur:in ausübt, wird genannt. </br>
<b>ACHTUNG</b>: Bei mehrdeutigen Bezeichnungen, wie Künstler:in oder Autor:in ist abzuwägen, ob es sich um einen Beruf, eine Tätigkeitsbeschreibung oder einen Verweis auf bestimmte Fähigkeiten handelt.</div></p>
<p><div style="margin-left:20px;"><b>3</b> Tätigkeitsbeschreibung </br>
Der Bereich, in dem der:die Akteur:in tätig ist, oder die Aufgabe, die er:sie (im vorliegenden Kontext) übernimmt, wird angesprochen. </br>
Beispiel: Ilya Sutskever, der sich mit der Weiterentwicklung von Künstlicher Intelligenz befasst.</br>
Auch vorangestellte "Kurzbeschreibungen" der Tätigkeit, wie <b>KI</b>-Experte, <b>Konstruktions</b>-Profi etc. reichen aus, um diesen Code zu vergeben.</br>
<b>ACHTUNG</b>: Werden hauptberuflich nicht als Schriftsteller:innen tätige Personen als Autor:innen bezeichnet (z. B. Wissenschaftler:innen, die als Studienautor:innen auftreten; Prominente, die eine Biografie verfasst haben, usw.), wird die Bezeichnung "Autor:in" als Tätigkeitsbeschreibung in dieser Kategorie codiert und NICHT unter "2 = Berufsbezeichnung" erfasst. </div></p>
<p><div style="margin-left:20px;"><b>4</b> Verweis auf Qualifikation, Expertise und/oder (außergewöhnliche) Fähigkeiten </br>
Die Ausbildung, Kompetenz, Wissen oder andere Kriterien, die darauf hindeuten, dass der:die Akteur:in in besonderem Maße (für eine bestimmte Sache) qualifiziert ist, werden erwähnt.</br>
Beispiel: Der Experte Ilya Sutskever, Die Affenflüstererin Jane Goodall, Der Überlebenskünstler Felix Baumgarnter</br>
<b>ACHTUNG</b>: Nennungen von akademischen Titeln, Auszeichnungen o. Ä. werden NICHT in dieser Kategorie, sondern unter "5 = Titelnennung" erfasst.</div></p>
<p><div style="margin-left:20px;"><b>5</b> Titelnennung</br>
Akademische Grade, Auszeichnungen, Preise oder ähnliche leistungsbezogene und durch Dritte verliehene Titel werden genannt.</br>
Beispiele: Prof. Christian Drosten; Medizinnobelpreisträger Gregg Semenza</div></p>
<p><div style="margin-left:20px;"><b>99</b> sonstige Beschreibung oder Bezeichnung </br>
Dieser Code wird nur ausgewählt, wenn keine der Auswahlmöglichkeiten zutreffend ist, und ein:e Akteur:in bei ihrem ersten Auftreten nicht nur namentlich genannt wird.</div></p>
<p><div style="margin-left:20px;"><b>0</b> nur Name</br>
Die Person wird ausschließlich mit ihrem Namen vorgestellt.</div></p>
